# Tic Tac Toe With AI
Features:
 - Responsive Swing UI with smooth hover and press animations
 - Themes (Light / Dark / Calm) and gestures (hover highlight, click ripple)
 - Minimax AI with alpha-beta pruning and optional depth limit (difficulty)
 - Move ordering (center, corners, edges) to speed up search
 - Undo, Restart, New Game options
 - Clear separation of Model (Board), AI, and View (Swing UI) logic

Compile and run:
 javac TicTacToeAI.java
 java TicTacToeAI

Tested on Java 11+. No external libraries required.
